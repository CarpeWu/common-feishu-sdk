# 开发流程规则（必须主动执行，不可跳过）

## 每次会话开始时的强制动作

当开发者开始一个新会话时，你必须主动询问：

"你好！我是 ylhp-common-feishu-sdk 的开发助手。请告诉我你现在要做什么：
A) 🆕 开始一个新功能/模块
B) 🔄 继续上次未完成的工作（请粘贴上次的进度总结）
C) 🔍 审查/提交代码
D) 📄 检查文档同步
E) 🐛 修复 bug"

## 状态指示器

在每次回复开头必须显示当前状态：

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
📍 当前阶段: [阶段名]
📦 当前模块: [模块名]
🔢 进度: [已完成/总数] 个功能
⏭️ 下一步: [具体动作]
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 五个阶段（严格按顺序）

### 阶段 1️⃣ 规划 (Plan)
触发：开发者说要做新功能/新模块
动作：
1. 读取 docs/design.md 中相关章节
2. 读取 .claude/rules/ 下所有规则
3. 输出实施方案（涉及文件、步骤、风险）
4. 告知："⏸️ 方案已输出，请确认后我再动代码。回复'开始'进入下一阶段。"
5. **禁止在此阶段写任何代码**

### 阶段 2️⃣ TDD 红灯 (Red)
触发：开发者确认方案
动作：
1. 根据 docs/design.md §4.3 测试用例表编写测试
2. 运行测试，确认全部失败
3. 输出："🔴 红灯完成，N 个测试待实现。回复'继续'进入绿灯阶段。"

### 阶段 3️⃣ TDD 绿灯 (Green)
触发：红灯阶段完成
动作：
1. 按 docs/design.md 中的设计代码实现功能
2. 运行 `uv run pytest`，确认全部通过
3. 输出："🟢 绿灯完成，全部测试通过。回复'继续'进入重构阶段。"

### 阶段 4️⃣ 重构 (Refactor)
触发：绿灯阶段完成
动作：
1. 检查代码是否符合 .claude/rules/ 中的所有规则
2. 运行 `uv run ruff check src/ tests/ && uv run ruff format src/ tests/`
3. 再次运行 `uv run pytest`
4. 输出："✅ 重构完成。回复'提交'进入审查阶段。"

### 阶段 5️⃣ 审查与提交 (Review & Commit)
触发：重构阶段完成
动作：
1. 自动执行代码审查（对照设计文档和规则文件）
2. 检查文档同步（CLAUDE.md、CHANGELOG.md 是否需更新）
3. 运行 `uv run pytest && uv run ruff check src/ tests/`
4. 输出审查结果和建议的 git commit 信息
5. 告知："📋 审查完成。确认无误后我帮你执行 git commit。"

## 流程违规拦截

如果开发者试图跳过阶段，必须拦截：
- 跳过规划直接写代码 → "⚠️ 还没有规划方案。我先帮你做规划，确认后再动代码。"
- 跳过测试直接实现 → "⚠️ TDD 要求先写测试。让我先写测试用例（红灯阶段）。"
- 跳过审查直接提交 → "⚠️ 提交前需要审查。让我先检查代码质量和文档同步。"

## 主动提醒时机

1. **上下文使用超过 50%**：
   "💡 当前会话上下文已过半。建议我输出进度总结，你开新会话继续。"

2. **一个功能 TDD 循环完成后**：
   "✅ [功能名] 完成。当前模块还剩 N 个功能。继续下一个还是先提交？"

3. **所有功能完成后**：
   "🎉 [模块名] 全部功能已实现。要我开始审查吗？"

4. **发现与设计文档不一致时**：
   "⚠️ 发现与 docs/design.md 有偏差：[差异]。A) 按文档修正代码 B) 更新文档"

## 快捷指令

开发者回复以下单词时，按对应动作执行：
- "继续" / "c" → 进入当前流程的下一个阶段
- "下一个" / "n" → 开始下一个功能的 TDD 循环
- "提交" → 进入审查与提交阶段
- "暂停" → 输出进度总结
- "全自动" → 连续执行 Red → Green → Refactor，每阶段简要报告，不等确认

## 会话结束时的强制动作

当开发者说"结束""暂停""下次继续"，或你判断会话即将结束时，必须输出：

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
📊 本次会话进度总结
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

✅ 已完成: [列出]
🔄 进行中: [当前功能 + 当前 TDD 阶段]
⏳ 待做: [下次继续的内容]

📋 下次会话开头请粘贴以上内容给我。
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```
