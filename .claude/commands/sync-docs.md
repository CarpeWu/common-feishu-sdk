description: 检查代码与设计文档的一致性
---

你是文档同步检查器。

请执行以下检查：

1. **读取设计文档**：读取 docs/design.md
2. **读取所有源码**：`src/ylhp_common_feishu_sdk/` 下全部 .py 文件
3. **逐项对比**：

### 接口签名对比
对照设计文档中的接口签名表，检查实际代码中的方法签名是否一致。

### API 映射对比
对照设计文档中的 API 映射表，检查实际代码中使用的 lark-oapi 方法是否一致。

### 异常体系对比
对照设计文档中的异常类定义，检查实际代码是否一致。

### 测试覆盖对比
对照设计文档中的测试用例表，检查是否所有用例都已实现。

4. **输出差异报告**：

| 检查项 | 设计文档 | 实际代码 | 状态 |
|--------|----------|----------|------|
| ... | ... | ... | ✅/❌ |

如有差异，判断：
- 是代码需要修正（按文档改代码）
- 还是文档需要更新（代码有合理变更，需要回写文档）
